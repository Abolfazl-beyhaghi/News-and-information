کی از روش‌هایی که برای جبران کمبود ویتامین C می‌توان استفاده کرد تزریق آمپول ویتامین C است. آمپول ویتامین C تایید سازمان غذا و داروی آمریکا را دارد. آمپول ویتامین C برای کمک به درمان زخم‌های جدی ناشی از ضربه و سوختگی مفید و موثر است. جذب ویتامین C از طریق تزریق سرعت بالاتری نسبت به مکمل‌های خوراکی دارد و به همین علت روند درمان با سرعت بالاتری طی می‌شود از فواید آمپول ویتامین C می‌توان به مواردی مانند کمک به درمان سرطان، تقویت عملکرد سیستم ایمنی، روشن شدن پوست، پیشگیری از چین و چروک پوستو کاهش وزن سریع اشاره کرد. تزریق آمپول ویتامین C در دوران بارداری فقط و فقط باید با دستور پزشک باشد. همچنین افراد مبتلا به بیماری ‌های کلیوی نیز برای تزریق آمپول ویتامین C باید با دکتر خود مشورت کنند. در مورد عوارض و مضرات احتمالی ویتامین C می‌توان از مواردی مانند سردرد، سوزش سر دل، تهوع، اسهال، استفراغ، خواب آلودگی و یا برعکس بیخوابی و گرگرفتی پوست یاد کرد.
